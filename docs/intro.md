# Postlane Documentation

Postlane turns your development workflow into short and long-form content. Write code, commit it, and let Postlane draft the posts.

## How it works

1. Initialise a repo with `npx @postlane/cli init` and add it to the desktop app
2. Run `/draft-post` in your IDE -- Postlane reads your git context and drafts posts based on what changed
3. Edit the draft, customise per platform, and approve in the desktop app -- or schedule for a later time via a connected provider

## What you can post to

LinkedIn, X, Bluesky, Mastodon, Instagram, Facebook, Pinterest, Substack Notes, and more. The full platform list depends on which [scheduling provider](/scheduling) you connect -- Zernio alone covers X, LinkedIn, Bluesky, Mastodon, Instagram, Facebook, and Pinterest.
