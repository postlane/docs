# Analytics

Postlane tracks engagement for posts sent through the app — likes, shares, comments, and impressions — and surfaces them in the desktop app's Analytics view.

## How it works

After a post goes live, Postlane's daily sync fetches engagement data from the platform (where the API permits) and stores it locally. You can see per-post performance and spot trends over time without leaving the app.

## Viewing analytics

In the desktop app, select a repository and click the **Analytics** tab. You will see a timeline of posts with engagement metrics alongside each one.
