# Zernio

Zernio is the recommended scheduling provider for Postlane. If you are on Postlane Pro, a scheduling allocation is bundled — you do not need a separate Zernio account or subscription.

## Setup

1. In the desktop app, go to **Settings > Scheduler**
2. Select **Zernio**
3. Your bundled allocation is activated automatically on sign-in — no API key required for Pro users
