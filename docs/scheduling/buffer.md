# Buffer

Buffer is a scheduling provider you can connect if you already have an existing Buffer account. Postlane does not bundle Buffer access — you need a paid Buffer subscription separately.

:::info
If you do not already have a Buffer account, [Zernio](/scheduling/zernio) is included in Postlane Pro at no extra cost.
:::
